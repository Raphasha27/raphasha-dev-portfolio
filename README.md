## Overview

Monorepo containing my professional portfolio ecosystem:

| Package | Description |
|---------|------------|
| `apps/portfolio-site` | Main portfolio website |
| `apps/api` | Backend API services |
| `apps/admin-ui` | Admin dashboard |
| `packages/ui-components` | Shared UI component library |
| `packages/utils` | Shared utility functions |

## Getting Started

```bash
# Each app has its own setup instructions
cd apps/portfolio-site
# Follow app-specific README
```

## Structure

```
raphasha-dev-portfolio/
```
